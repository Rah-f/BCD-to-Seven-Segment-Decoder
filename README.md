# SevenSegment_Decoder_Display
BCD to Seven-Segment Decoder using Karnaugh Maps, Boolean Simplification, and Logic Circuit Design.

# BCD to Seven-Segment Decoder

This repository contains the complete design and implementation of a BCD-to-Seven-Segment Decoder using combinational logic design techniques.

# Project Contents

* BCD to Seven-Segment Truth Table.
* Karnaugh Maps (K-Maps) for each output segment (a–g).
* Simplified Boolean expressions obtained from the K-Maps.
* Logic circuit design for each output segment.
* Final integrated decoder circuit combining all segment outputs.
* Enhanced circuit layout with an improved circular design.

# Objective

The objective of this project is to convert BCD (Binary-Coded Decimal) inputs into the corresponding outputs required to drive a seven-segment display while minimizing the logic using Karnaugh Maps and Boolean algebra.

# Files Included

* Truth Table
* Karnaugh Maps
* Boolean Expressions
* Logic Circuits
* Final Decoder Circuit
